Bootstrap nav-wizard

This project adds a .nav-wizard CSS class to complement Bootstrap's OOTB .nav-tabs and .nav-pills classes. It follows the same API as those two classes: you need only set the .active class on the active step, and preceding steps will be styled as complete, while future steps will be styled as not yet completed.

Project Info

Project Goals

  • Mimic the existing bootstrap nav-* class APIs as closely as possible
  • Provide a pure, dependency-other-than-bootstrap-free LESS/CSS implementation
  • Reasonably flexible for use in either client-side javascript or server-side wizard workflows
  • Approximate the look and feel of the Fuel UX wizard classes
