Bootstrap Wizard Navigation

A pure Less/CSS wizard navigation element which behaves like tabs/nav-pills

Goals

  • Identical API to nav-pills and nav-tabs
  • Reasonably flexible for use in either client-side javascript or server-side wizard workflows
  • Pure Less/CSS implementation
  • Look and Feel like the Fuel UX wizard
